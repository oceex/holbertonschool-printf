##_printf

A project recreating printf function. 

## Description

_printf function outputs using a format string with conversion specifiers.

## Specifiers

- %c  - character
- %s - string
- %d  - integers
- %i  - integers
- %o  - octal
- %x  - hex
- %X  - HEX
- %b  - binary
- %r  - reversed string
- %R - rot13 string

## Compilation

$ gcc -Wall -Werror -Wextra -pedantic -std=gnu89 -Wno-format *.c
